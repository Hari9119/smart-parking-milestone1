#  Smart Parking System – Milestone 1

##  Project Overview
The **Smart Parking System** is a full-stack web application designed to manage and visualize parking slot availability.  
This repository contains the **Milestone 1 implementation**, focusing on backend setup, database integration, and frontend connectivity.

---

##  Milestone 1 Objectives
✔ Express server setup  
✔ MongoDB database connection using Mongoose  
✔ Parking Slot data model  
✔ Slot seeding script for initial data  
✔ REST API to fetch parking slots  
✔ React frontend to visualize parking slots  
✔ Color-coded UI (Available / Booked)

---

##  Tech Stack
**Backend**
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

**Frontend**
- React (Vite)
- CSS for styling
